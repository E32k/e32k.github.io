# Vehicle Lua (vlua)

Vehicle Lua runs **per-vehicle** and is performance-sensitive.

## Capabilities

- Engine simulation
- Powertrain logic
- Sensors

## Inline Example

Use `obj:getVelocity()` to read vehicle speed.

## Folder behavior

This page represents the **vlua folder**.
Clicking `vlua` in the sidebar lands here.
